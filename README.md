#  Basic Todo List App (React)

A simple and functional *Todo List App* built using *React*.  
This project focuses on *React fundamentals* — state management, event handling, and rendering lists — without any external styling yet.

---

##  Features

- ➕ *Add Todo:* Type a task in the input box and click the *Add* button to create a todo.  
- 🗑️ *Delete Todo:* Click the delete button to mark a todo as *completed* — it will appear with a line-through effect.  
- 🔄 *Dynamic Rendering:* Todos are rendered dynamically using the .map() method.  
- ⚡ *React Hooks:* Uses useState to manage todos in real time.

---

##  What I Learned

- Handling *state* in React using useState.
- Managing *input fields* and *button events*.
- Rendering lists dynamically with the *map()* function.
- Updating UI reactively based on state changes.
- Understanding the *concept of immutability* in React arrays.
